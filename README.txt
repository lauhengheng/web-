项目名称：仿MCAKE电商小程序（此项目是在web程序设计课程中本人的期末程序设计）
创作时间：2022.5

1.这个项目是本人仿MCAKE蛋糕商城(https://h5.mcake.com)完成的一个项目（是一个卖蛋糕的移动端电商程序），通过控制台查看html结构和css
样式及js完成了大部分效果，也是利用hbuilderx设计的。

2.其中主要有首页，分类，购物车，我的等页面，其他页面包括商品分类页面，商品详情页，分类导航栏等。实现了注册，登录，添加购物车等功能。
